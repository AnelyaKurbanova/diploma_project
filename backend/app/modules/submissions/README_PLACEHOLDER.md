This module is reserved for future implementation of submissions, progress tracking, and teacher rooms.

Planned responsibilities:

- Map `submissions` and `submission_choice_map` tables to SQLAlchemy models.
- Provide services for students to submit answers and receive grading results.
- Expose APIs for teachers to see student attempts and progress (later tied to rooms).

No runtime code is implemented yet; this file documents the intended boundary so that
current catalog and problems modules can evolve without breaking future plans.
